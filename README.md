## 🍷 Wine Quality Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting the **quality of wine** based on its chemical properties using **Machine Learning algorithms**. Wine quality prediction helps producers and researchers understand which chemical factors influence wine quality and how machine learning can assist in classification tasks.

This project was completed as part of the **Oasis Infobyte Data Analytics Internship (OIBSIP)**.

---

## 🎯 Objective

The main objective of this project is to build a **machine learning classification model** that predicts the quality of wine using its chemical attributes.

The project also demonstrates how data analytics and machine learning can be applied to solve real-world problems in the food and beverage industry.

---

## 📂 Dataset

The dataset used in this project is **WineQT.csv**, which contains various chemical properties of wine samples along with their quality ratings.

### Key Features in Dataset

* Fixed Acidity
* Volatile Acidity
* Citric Acid
* Residual Sugar
* Chlorides
* Free Sulfur Dioxide
* Total Sulfur Dioxide
* Density
* pH
* Sulphates
* Alcohol
* Quality (Target Variable)

Each row represents a wine sample with its chemical characteristics and quality score.

## ⚙️ Project Workflow

### 1. Data Loading

The dataset was loaded using the **Pandas** library.

### 2. Data Exploration

Basic dataset information and statistical summaries were analyzed to understand the dataset structure.

### 3. Data Visualization

Seaborn and Matplotlib were used to visualize relationships between features and understand patterns in the dataset.

### 4. Feature Selection

All chemical properties were used as input features for predicting wine quality.

### 5. Train-Test Split

The dataset was divided into **training data (80%)** and **testing data (20%)**.

### 6. Model Training

Machine learning models such as **Random Forest Classifier** were used to train the dataset.

### 7. Prediction

The trained model predicts the **quality level of wine** based on the given chemical features.

### 8. Model Evaluation

The model was evaluated using **accuracy score** and **classification report**.
